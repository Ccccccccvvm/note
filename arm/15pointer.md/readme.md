## 加减运算+/-

![pointer01](https://cdn.jsdelivr.net/gh/nzcv/picgo/pointer01.png)


总结：

1、带*类型的变量可以加、减一个整数，但不能乘或者除；

2、带*类型变量与其他整数相加或者相减时：

    带*类型变量 + N = 带*类型变量 + N*(去掉一个*后类型的宽度)；
    
    带*类型变量 - N = 带*类型变量 - N*(去掉一个*后类型的宽度)。
